הרפתקת ניתוח נתוני משחקים

📊 סקירת הפרויקט

ברוכים הבאים להרפתקת נתונים אפית דרך הממלכה המיסטית של ניתוח הכנסות משחקים! הניתוח המקיף הזה הופך נתוני תשלום גולמיים לבינה עסקית מעשית, חוקר את המסעות הפיננסיים של 383 לוחמי משחקים אמיצים שתרמו 63,141$ לממלכה הדיגיטלית שלנו לאורך 2022.

כמו אביר נתונים מיומן המנווט במבוכים מסוכנים של מערכי נתונים מורכבים, הפרויקט הזה מפענח את הסודות החבויים בדפוסי תשלום של שחקנים, חושף תובנות אגדיות על התנהגות משתמשים ומגלה אוצרות של הזדמנויות אופטימיזציה להכנסות.

🎯 יעדי המשימה
  • מיומנות SQL מתקדמת: ארכיטקטורת 3-CTE מורכבת עם פונקציות חלון LAG/LEAD
  • מודיעין הכנסות: ניתוח MRR מלא (חדש, נטוש, הרחבה, כיווץ)
  • סיפור אינטראקטיבי: לוח מחוונים Tableau עם 6 הדמיות משכנעות
  • תובנות דמוגרפיות: פילוח משתמשים לפי גיל וניתוח התנהגות
  • אסטרטגיה עסקית: המלצות מעשיות לצמיחה ושימור
📊 תגליות מפתח

משלחת הנתונים שלנו חשפה תובנות יוצאות דופן מממלכת המשחקים:

  • 383 לוחמי משחקים תרמו לאוצר הממלכה שלנו
  • 63,141$ הכנסה כוללת שנוצרה לאורך המסע האפי של 2022
  • 165$ ARPPU ממוצע (הכנסה ממוצעת למשתמש משלם)
  • 14,186$ MRR חדש מגיבורים חדשים אמיצים המצטרפים לממלכה שלנו
  • 144.72$ ערך לקוח לכל החיים המראה את הערך האמיתי של כל לוחם
🗡️ מדדי קרב הכנסות

הניתוח חושף ארבעה עמודי הכנסה קדושים:

  • MRR חדש: גיבורים טריים המצטרפים לממלכה שלנו (14,186$ התגלו)
  • הכנסה נטושה: לוחמים עוזבים את הממלכה שלנו (דפוסים עונתיים התגלו)
  • הכנסת הרחבה: גיבורים מתחזקים (שיא בחודשי הקיץ)
  • הכנסת כיווץ: התאמות כוח אסטרטגיות (דפוסי חורף)
🧙‍♂️ כישוף טכני

כישופי SQL בשימוש:

  • ביטויי טבלה נפוצים מורכבים (CTEs) להפיכת נתונים
  • פונקציות חלון LAG/LEAD לניתוח מסע בזמן
  • מניפולציה של תאריכים לצבירת הכנסות חודשית
  • הצהרות CASE מתקדמות לקטלוג הכנסות

כישופי הדמיה:

  • ניתוח מגמת הכנסות חודשית
  • פירוק מפל הכנסות
  • דפוסי שיעור נטישת משתמשים
  • מעקב התנהגות ARPPU
  • תובנות ערך לקוח לכל החיים
🏆 השפעה עסקית ותובנות אסטרטגיות
  • גילוי רכישת שיא: אוגוסט הופיע כחודש הזהב עם 50 לוחמים חדשים
  • דפוסים עונתיים: זוהה זינוק נטישה של 100% בדצמבר הדורש אסטרטגיות שימור
  • צמיחת הכנסות: מסע מרהיב מ-1,491$ (מרץ) ל-9,344$ (אוקטובר)
  • מודיעין דמוגרפי: קטעים מבוססי גיל המראים דפוסי מונטיזציה נבדלים
🛠 טכנולוגיות בשימוש
  • PostgreSQL: SQL מתקדם עם פונקציות חלון ו-CTEs
  • Tableau Public: יצירת לוח מחוונים אינטראקטיבי וסיפור נתונים
  • HTML5/CSS3: פיתוח אתרים רספונסיבי להצגת פרויקט
  • GitHub Pages: פריסה ואירוח פרויקט מקצועי
